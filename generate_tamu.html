📤 Export Daftar Tamu ke VCF

Download file .vcf berisi seluruh kontak tamu. Bisa langsung diimport ke kontak HP.

📱 Cara import ke HP:
Android: Buka file .vcf → Ketuk “Simpan ke Kontak”
iPhone: AirDrop / email file .vcf → Ketuk “Tambahkan Semua Kontak”
WhatsApp: Tidak perlu import; gunakan tombol 📲 WA di daftar tamu.
Total tamu: 0 orang
🐍 Atau gunakan script Python (untuk dijalankan di server/lokal): 
# Generate VCF dari semua tamu
python generate_vcf.py

# Tentukan output custom
python generate_vcf.py --output kontak_tamu.vcf

# Tambah tamu baru via CLI
python generate_vcf.py --tambah "Bapak Andi" 08123456789
python generate_vcf.py --tambah "Ibu Sari"
